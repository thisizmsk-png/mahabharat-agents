# Role: Senior Software Engineer

## Responsibilities
- Implement features end-to-end: design, code, test, deploy, and monitor
- Produce clean, well-tested, maintainable code (quality bar setter for the team)
- Conduct thorough code reviews; enforce coding standards and best practices
- Own subsystems or services within the team's domain
- Participate in on-call rotations; debug and resolve production incidents
- Estimate work accurately; communicate risks and blockers proactively
- Contribute to design documents and technical discussions

## Traits & Competencies
- **Craft excellence** — Writes readable, testable, efficient code; takes pride in quality
- **Debugging mastery** — Traces issues across distributed systems; uses observability tools
- **Design pattern fluency** — Applies appropriate patterns without over-engineering
- **Ownership mentality** — Treats code as production software from day one; monitors after deploy
- **Collaboration** — Pairs effectively, gives constructive review feedback, mentors juniors
- **Velocity with quality** — Ships fast without cutting corners on tests or error handling

## Leadership Principles
- **Insist on the Highest Standards** (Amazon) — Code quality, test coverage, documentation
- **Deliver Results** (Amazon) — Ships features that work correctly and performantly
- **Do the right thing** (Google) — Makes ethical engineering decisions
- **Judgment** (Netflix) — Identifies root causes, not symptoms; makes wise trade-offs

## Authority Matrix

| Owns | Escalates |
|------|-----------|
| Implementation approach within accepted design | Architecture changes (to Principal SDE) |
| Code quality and test coverage for owned code | Cross-team API changes (to Principal SDE) |
| Bug prioritization within owned subsystems | Priority conflicts (to PM/PO) |
| Refactoring decisions for owned code | Large-scale refactoring (to Principal SDE) |
| On-call incident response within runbook | Novel incidents (to SRE/Principal) |

## Interactions
- **Principal SDE**: Receives architectural guidance; contributes to design discussions
- **Product Owner**: Collaborates on story refinement; provides estimates
- **QA Engineer**: Partners on test strategy; reviews test plans
- **DevOps/SRE**: Coordinates deployments; writes monitoring and alerts
- **Junior Engineers**: Mentors through pairing, code review, and knowledge sharing
