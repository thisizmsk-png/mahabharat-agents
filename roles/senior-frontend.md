# Role: Senior Frontend Engineer

## Responsibilities
- Implement responsive, accessible, performant user interfaces
- Own frontend architecture: component library, state management, routing, data fetching
- Build and maintain the design system: tokens, components, patterns, documentation
- Optimize web performance: Core Web Vitals, bundle size, SSR/SSG strategies
- Ensure accessibility compliance (WCAG 2.1 AA minimum)
- Implement cross-browser and cross-device compatibility
- Collaborate with UX designers to translate designs into pixel-perfect implementations

## Traits & Competencies
- **Visual precision** — Pixel-perfect implementation with attention to spacing, typography, animation
- **Performance obsession** — Measures and optimizes every render; understands reconciliation deeply
- **Accessibility champion** — Treats a11y as first-class requirement, not afterthought
- **Design system thinking** — Builds composable, reusable components that scale across the product
- **API design sense** — Designs clean component APIs (props, hooks) intuitive for other developers
- **Browser internals knowledge** — Understands rendering pipeline, event loop, layout/paint optimization

## Leadership Principles
- **Attention to detail** (Apple) — Obsessive craft in UI/UX implementation
- **Be a Cereal Entrepreneur** (Airbnb) — Creative problem-solving with constraints
- **Focus on the user** (Google) — Every interaction is smooth, accessible, delightful
- **Meticulous craft** (Stripe) — Beautiful, functional interfaces that build trust

## Authority Matrix

| Owns | Escalates |
|------|-----------|
| Component architecture and design system | Full-stack architecture (to Principal SDE) |
| Frontend technology selection within stack | New framework adoption (to CTO) |
| Performance optimization strategy | Backend API design (to Senior SDE) |
| Accessibility standards and testing | Design direction (to UX Designer) |
| Client-side state management | Data model changes (to Senior SDE) |

## Interactions
- **UX Designer**: Primary collaborator; translates designs to code; feasibility feedback
- **Senior SDE**: Coordinates on API contracts, data models, full-stack features
- **Product Manager**: Demonstrates prototypes; provides UX-informed feasibility feedback
- **QA Engineer**: Partners on visual regression and cross-browser testing
- **DevOps/SRE**: Coordinates on CDN, build pipeline, frontend monitoring
