## Zero Cognitive Bias Protocol (MANDATORY)

Every agent, every decision, every recommendation must be free of:

1. **Anchoring** — Evaluate independently, not relative to first option seen
2. **Confirmation** — Actively seek DISCONFIRMING evidence
3. **Availability** — Use data, not memory of recent/memorable examples
4. **Survivorship** — Include failures, not just successes
5. **Authority** — Popular ≠ correct. Evaluate on merit.
6. **Recency** — Give appropriate weight to historical data
7. **Sunk Cost** — Evaluate current merit, not past investment

## 5-Question Research Framework (MANDATORY for non-trivial decisions)

1. **Domain Understanding** — Core concepts and terminology
2. **Business Rules** — Governing rules, invariants, constraints
3. **Industry Standards** — Established patterns and best practices
4. **Data Requirements** — Structures, models, relationships
5. **Edge Cases** — Complex, unusual, or failure scenarios

## Authority Rules

2. Destructive operations require Krishna (CEO) approval
3. Security concerns escalate to Bhishma regardless of hierarchy
4. Deadlocked debates escalate to Krishna for final decision
5. Sepoys escalate all design decisions to their master
6. Every decision must be logged with reasoning

## Communication Protocol

- Agents communicate via **typed artifacts**, not free-form chat
- Every artifact has: author, type, content, confidence, reasoning, timestamp
- Downstream agents validate artifacts before consumption
- Full audit trail maintained for every agent interaction
