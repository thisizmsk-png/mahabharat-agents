# Credits & Attribution

The Mahabharat Agents Package stands on the shoulders of giants. We gratefully acknowledge
the following projects whose ideas, patterns, and research informed this framework:

## Frameworks That Shaped Our Architecture

### MetaGPT
- **Repository:** https://github.com/geekan/MetaGPT
- **License:** MIT
- **What we learned:** SOP-as-code pattern, artifact-based pub-sub communication,
  structured role definitions. The insight that `Code = SOP(Team)` is foundational.

### CrewAI
- **Repository:** https://github.com/crewAIInc/crewAI
- **License:** MIT
- **What we learned:** The Role-Goal-Backstory trinity for agent definition.
  Our Dharma-Karma-Katha format is directly inspired by this elegant pattern.

### ChatDev
- **Repository:** https://github.com/OpenBMB/ChatDev
- **License:** Apache 2.0
- **What we learned:** Phase-based lifecycle for software development,
  paired agent communication patterns.

### Swarms
- **Repository:** https://github.com/kyegomez/swarms
- **License:** MIT
- **What we learned:** Multi-topology orchestration. The insight that no single
  pattern fits all use cases — sequential, hierarchical, mixture, graph all have
  their place.

### MiroFish
- **Repository:** https://github.com/666ghj/MiroFish
- **License:** AGPL-3.0
- **What we learned:** God's-perspective visualization for multi-agent systems.
  The Kurukshetra dashboard concept is inspired by MiroFish's real-time agent
  monitoring with variable injection.

### OpenViking
- **Repository:** https://github.com/volcengine/OpenViking
- **License:** Apache 2.0
- **What we learned:** Tiered context management (L0/L1/L2). Our Shruti/Smriti/Itihasa
  context hierarchy is directly inspired by OpenViking's approach to managing
  agent memory efficiently.

### AutoGen
- **Repository:** https://github.com/microsoft/autogen
- **License:** MIT
- **What we learned:** Conversable agent abstraction, nested chat patterns,
  group chat with configurable turn management.

### CAMEL
- **Repository:** https://github.com/camel-ai/camel
- **License:** Apache 2.0
- **What we learned:** Inception prompting for deep context establishment,
  workforce/channel organizational patterns.

### OpenHands
- **Repository:** https://github.com/OpenHands/OpenHands
- **License:** MIT
- **What we learned:** Event-sourced state model with deterministic replay
  for debugging and audit trails.

## Cultural Inspiration

The Mahabharata — one of humanity's greatest epics — provides the naming,
personality framework, and organizational metaphor for this project. Each agent's
character traits, decision-making style, and interpersonal dynamics are drawn
from the rich tapestry of the Mahabharata's characters. This is not just a naming
convention — it is a design philosophy where each agent's mythological persona
genuinely influences its behavior and judgment.
